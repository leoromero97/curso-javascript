Clase 22

Crear objetos literales
  • - Casi todo en JavaScript es un objeto.
  • - Se representan entre llaves.
  • - Pueden tener diferentes valores y cada valor puede ser cualquier tipo de dato.
Ejemplos:
// Creando un objeto literal const profile = { name: "Leonardo", price: 2000, isAvailable: false, };
console.log("Ver objeto profile:", profile);
// Creando un objeto constructor function Profile(name, price) { this.name = name; this.price = price; this.isAvailable = true; }
const profile2 = new Profile("Esteban", 1300);
console.log("Mostrar perfil 2 con objeto constructor:", profile2);

Clase 23

Accediedo a valores de un objeto
  • - Los valores de un objeto serían las propiedades del mismo.
Ejemplos:
// Primera forma const profile1 = { lastName: "Romero", price: 2200, available: true, };
console.log( "Como acceder a un valor específico de un objeto:", profile1.lastName );
// Segunda forma console.log( "Como acceder a un valor específico de un objeto de otra forma:", profile1["lastName"] );

Clase 24

Modificando valores de un objeto
  • - Se pueden agregar, cambiar y eliminar propiedades a un objeto.
Ejemplos:
// Agregando un valor nuevo const productObject = { name: "Monitor 20 pulgadas", price: 70000, available: true, };
productObject.image = "image.png";
console.log(productObject);
// Cambiando un valor existente productObject.available = false;
console.log(productObject);
// Eliminando un valor existente delete productObject.price;
console.log(productObject);

Clase 25

Destructiring de un objeto
  • - Se pueden obtener los valores de un objeto a partir de la destructuración del mismo.
Ejemplos:
// Creando una constante a partir de un valor const productObject2 = { productName: "MacBookc Pro", price: 3000, available: true, };
const nombre = productObject2.productName;
console.log("Constante creada a partir de un valor de un objeto:", nombre);
// Destructurando un valor de un objeto const { productName } = productObject2;
console.log("Valor destructurado", productName);

Clase 26

Objetos dentro de objetos
  • - Un objeto puede tener múltiples objetos dentro del mismo.
  • - Se puede destructurar objetos anidados.
Ejemplos:
// Acceder al objeto dentro de un objeto const productObject3 = { productName: "Aire acondicionado", price: 800000, available: true, information: { weight: "1kg", height: "50cm", width: "70cm", }, };
console.log(productObject3.information);
// Acceder al valor de un objeto dentro de un objeto console.log(productObject3.information.weight);
// Acceder al valor de un objeto anidado por destructuring: const { productName, information, information: { width }, } = productObject3;
console.log(productName, information, width);

Clase 27

Congelar objetos para no mutarlos
  • - Los objetos literales son de fácil mutación.
  • - No siempre vamos a querer modificar el objeto principal, si no sus valores.
  • - Usar el "use strict" hace que el código de JavaScript respete las buenas prácticas.
Ejemplos:
// Para congelar un objeto hay que habilitar el modo estricto const productObject4 = { productName: 'Tablet', price: 4000, available: true }
productObject4.image = 'image.png'
Object.freeze(productObject4)
productObject4.icon = 'icon.svg'
console.log("Usando el método freeze", productObject4)
// Sellar un objeto, no permite eliminar, ni agregar pero si permite editar propiedades existentes. const productObject5 = { productName: 'Celular', price: 1000, available: true }
productObject5.image = 'image.png'
Object.seal(productObject5)
productObject5.icon = 'icon.svg'
console.log("Esto no se renderiza porque no se puede agregar una propiedad usando el método seal:", productObject5)

Clase 28

Copiar 2 objetos
  • - Podemos unir uno o más objetos en uno solo.
Ejemplos:
// Ver objetos por separado const productObject6 = { productName: "Play Station", price: 900, available: true, };
const sizeProduct6 = { weight: "1kg", width: "15cm", };
console.log(productObject6)
console.log(sizeProduct6)
// Unificar objetos con método assign const objectsJoin = Object.assign(productObject6, sizeProduct6)
console.log('Objetos unificado', objectsJoin)
// Unificar objetos con spread operator const objectsJoin2 = {...productObject6, ...sizeProduct6}
console.log('Objetos unificado con spread operator', objectsJoin2)

Clase 29

Como acceder a propiedades internas
  • - Podemos acceder a propiedades internas de un mismo objeto con el método this.
Ejemplos:
// Crear una propiedad de tipo función (método del objeto) const productObject7 = { productName: "Play Station 5", price: 1200, available: true, getInformation: function () { console.log( `El producto ${this.productName} tiene un valor de $${this.price}` ); }, };
productObject7.getInformation();

Clase 30

Métodos útiles de objetos
  • - Existen diferentes métodos para usar en un tipo de dato objeto.
Ejemplos:
// Crear un objeto const productObject8 = { productName: "Play Station 4", price: 600, available: true, getInformation: function () { console.log( `El producto ${this.productName} tiene un valor de $${this.price}` ); }, };
console.log(productObject7);
// Obtener un array a partir de las propiedades de un objeto console.log( "Transformar un objeto a un array con los nombres de sus propiedades:", Object.keys(productObject8) );
// Obtener un array a partir de los valores de las propiedades de un objeto console.log( "Transformar un objeto a un array con los valores de sus propiedades:", Object.values(productObject8) );
// Obtener un objeto con arrays como propiedades, y cada array tiene su propiedad y su valor console.log( "Transformar un objeto a un objeto donde cada propiedad se vuelve un array con el nombre de la propiedad y su valor:", Object.entries(productObject8) );
