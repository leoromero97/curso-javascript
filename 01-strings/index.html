Clase 04

Fundamentos de Strings o cadenas de texto
  • - Es un texto, y este tiene que estar encerrado entre comillas.
  • - Se pueden utilizar comillas simples, dobles o inversas, es importante que se abran y cierren con el mismo tipo de comillas. '', "" o ``
  • - Pueden contener cualquier cosa, la intefaz siempre va a mostrar una cadena de texto siempre y cuando este dentro de las comillas.
  • - Se pueden concatenar/encadenar.
  • - Tienen métodos nativos de JavaScript.
  • - Si el valor es un string, se pueden utilizar comillas simples, dobles o inversas,
Ejemplos:
// Inicializar variables con un valores de tipo string: const producto1 = "Mochila";
console.log(producto1); const producto2 = 'Valija';
console.log(producto2); const producto3 = `Candado`;
console.log(producto3);

Clase 05

String Methods - Includes y length
  • - Los métodos son funciones nativas que probee JavaScript y hay diferentes para cada tipo de dato.
  • - También existen propiedades como .length
  • - Devuelven diferentes valores.
Ejemplos:
// Conocer la cantidad de letras de la cadena de texto: const product1 = 'Monitor 20 pulgadas'
console.log(product1) console.log('Cantidad de letras en la cadena de texto', product1.length)
// Conocer la posición de la letra donde empieza la palabra que buscamos. Si no se encuentra devuelve -1 const product1 = 'Monitor 20 pulgadas'
console.log(product1.indexOf('Monitor'))
// Devuelve un valor de tipo boolean para saber si incluye el string que le pasemos en el método console.log('Incluye tablet?', product1.includes('Tablet'))
console.log('Incluye Monitor?', product1.includes('Monitor'))
console.log('Incluye monitor?', product1.includes('monitor'))

Clase 06

Concatenar o unir strings
  • - Se puede utilizar el método concat().
  • - Se puede utilizar el signo + o , y espacios.
  • - Se puede utilizar template strings, con comillas invertidas, signo de $ y llaves {}.
Ejemplos:
// Concatenar con método concat() const product = 'Moto' const brand = 'Honda'
console.log(product.concat(' ', brand))
console.log(product.concat(' en descuento'))
// Concatenar con + y espacios console.log("La marca del producto: " + product + ", es " + brand)
// Concatenar con , y espacios console.log("El producto a vender es: " , product)
// Concatenar con template strings, se debe usar comilla invertida o bactick y signo de pesos ($) y encerrar entre llaves {} la variable console.log(`El producto es ${product} y su marca es ${brand}`)

Clase 07

Cortar espacios en blanco de un string
  • - Se puede utilizar el método trimStart() para cortar los espacios al inicio del string.
  • - Se puede utilizar el método trimEnd() para cortar los espacios al final del string.
  • - Se puede utilizar el método trim() o de forma concatenada los métodos trimStart() y trimEnd() para cortar los espacios al inicio y al final del string.
Ejemplos:
// Agregar espacios en blanco al principio y al final, JavaScript cuenta esos espacios como un carácter. const product4WithSpaces = " Botella de agua ";
console.log(product4WithSpaces.length);
// Eliminar espacios en blanco del inicio del string console.log(product4WithSpaces.trimStart());
// Eliminar espacios en blanco al final del string console.log(product4WithSpaces.trimEnd());
// Eliminar espacios en blanco al inicio y al final del string de forma concatenada console.log(product4WithSpaces.trimStart().trimEnd());
// Eliminar espacios en blanco al inicio y al final del string con 1 solo método console.log(product4WithSpaces.trim());
console.log(product4WithSpaces.trim().length);

Clase 08

Reemplazar o modificar valores del string
  • - Se puede utilizar el método replace() para reemplazar un valor por otro.
  • - Se puede utilizar el método slice() o substring() para cortar y extraer un valor del string en base a posiciones del index de ese string.
  • - Se puede utilizar el método chartAt() para obtener un valor del string en base a la posición que se pase.
Ejemplos:
// Reemplazar un string por otro con método replace() const product5 = 'Monitor de 20 pulgadas'
console.log(product5)
console.log(product5.replace('pulgadas', '"'))
console.log(product5.replace('Monitor', 'Monitor Curvo'))
// Método slice() para extraer o cortar una parte del string, hay que pasar el index del string donde quieres que inicie y donde quieres que corte console.log('Arranca en la posición 0, y desde la 10 en adelante lo corta al string: ', product5.slice(0, 10))
console.log('A partir de la posición 8 en adelante muestra el contenido del string: ', product5.slice(8))
console.log('Si la posición de inicio es más grande que la del index a cortar, no se ejecuta el método.', product5.slice(2, 1))
// Alternativa a método slice() con substring() console.log('Arranca en la posición 0, y desde la 10 en adelante lo corta al string: ', product5.substring(0, 10))
console.log('Si la posición de inicio es más grande que la del index a cortar, se invierten las posiciones: ', product5.substring(2, 1))
// Cortar la primera letra de un string const userName = 'Leo'
console.log(userName.substring(0,1))
console.log(userName.charAt(0))
// Eliminar espacios en blanco al inicio y al final del string con 1 solo método console.log(product4WithSpaces.trim());
console.log(product4WithSpaces.trim().length);

Clase 09

Repetir valores de un string y dividir un string
  • - Con el método repeat() se puede repetir un string la cantidad de veces que se indique.
  • - Con el método split() puedo obtener un array a partir del string.
  • - Puedo obtener un array por cada caracter del string o por cada palabra que este por fuera del valor que indique.
Ejemplos:
// repeat() permite repetir una cadena de texto const product6 = "Celular Xiaomi ";
const text = "en promoción ".repeat(3);
console.log(`${product6} ${text}!!!`);
// Si pasas un número que no es entero, se redondea al número entero, no tiene en cuenta si el decimal es mayor o menor a 5. const text2 = "en promoción ".repeat(2.7);
console.log(`${product6} ${text2}!!`);
// El método split() devuelve un array con cada caracter del valor del string. const activity = "Estoy aprendiendo JavaScript moderno";
console.log(activity.split(""));
// Devuelve un array con cada palabra del valor del string que este separado por un espacio. console.log(activity.split(" "));
// Devuelve un array con cada palabra del valor del string que este separado por el valor que le indique. const hobbies = "Leer, caminar, escuchar música, escribir, programar";
console.log(hobbies.split(", "));
const hashtag = "Aprendiendo JavaScript #JSModerno";
console.log(hashtag.split("#"));

Clase 10

Convertir a mayúsculas o minúsculas
  • - Con el método toUpperCase() se puede transformar todo el valor del string a mayúscula.
  • - Con el método toLowerCase() se puede transformar todo el valor del string a minúscula.
  • - Con el método toString() se puede convertir un valor de tipo number a tipo string.
Ejemplos:
// Pasar a mayúscula todo el valor del string const address = 'Castañon'
console.log(address.toUpperCase())
// Pasar a minúsculas todo el valor del string const email = 'Leonardogerbacio@gmail.com'
console.log(email.toLowerCase())
// Convertir un valor de tipo númerico a tipo string const price1 = 3000
console.log(price1)
console.log(price1.toString())
