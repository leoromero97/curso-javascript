Clase 31

Crear arrays/arreglos/listas
  • - Los arrays son listas en JavaScript.
  • - Agrupa varios elementos.
  • - Los elementos pueden ser iguales o distintos.
  • - Los elementos pueden ser cualquier tipo de dato, strings, numbers, booleans, objects, incluso, arrays.
  • - Los arrays también utilizan índices/posiciones para cada elemento.
Ejemplos:
// Crear un array con tipos de datos iguales const numbers1 = [10, 20, 30];
console.log(numbers1);
// Crear un array a partir del constructor de arrays const months1 = new Array("Enero", "Febrero", "Marzo");
console.log(months1);
// Crear un array con diferentes tipos de datos const arrayDiferents1 = [ "string 1", 20, false, { country: "Argentina" }, [1, 2, 3], ];
console.log(arrayDiferents1);

Clase 32

Acceder a arrays
  • - Es necesario conocer el índice del valor al que queremos acceder.
  • - Se utilizan corchetes y su índice para acceder a su valor.
Ejemplos:
// Acceder al valor 30 const numbers2 = [10, 20, 30, 40, 50, [1,2,3]];
console.log(numbers2);
console.table(numbers2);
console.log('Índice 2', numbers2[2]);
// Acceder al valor 2 del indice 5 console.log('Índice 5 y su sub índice 1', numbers2[5][1]);

Clase 33

Recorrer un array
  • - Es necesario conocer cuánto mide el array, cuántos elementos tiene.
  • - Se puede recorrer de diferentes maneras, la primera es el bucle/iterador for.
  • - En el iterador for maneja la posición de dónde quiero que inicie, cuántas veces queres que se ejcute el loop, y de a qué cantidad queres que se itere ese loop.

// Conocer la longitud del array const months2 = ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio"];
console.table(months2);

Clase 34

Agregar nuevos valores a un array
  • - Es necesario conocer cuánto mide el array, cuántos elementos tiene.
  • - Se pueden modificar los valores dentro de un array.
  • - Si agregas un elemento en una posición que no existe, JavaScript respeta esa posición que se le indica.
Ejemplos:
// Cambiar el valor de un elemento en un array const months3 = ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio"];
console.log(months3.length);
months3[0] = 'Mes inicial'
// Agregar un elemento en un array months3[7] = "Julio";
months3[10] = "Noviembre";
console.table(months3);

Clase 35

Agregar nuevos valores (de forma imperativa) al principio o final de un array
  • - Para agregar valores a un array podemos utilizar métodos.
  • - Se pueden agregar al principio o al final.
Ejemplos:
// Agregar un valor al final del array con el método push() const days1 = ["Lunes", "Martes", "Miércoles"];
console.table(days1);
days1.push("Jueves");
console.table(days1);
// Agregar un valor al inicio del array con el método unshift() days1.unshift("Domingo");
console.table(days1);

Clase 36

Crear un nuevo elemento con spread operator (forma declarativa)
  • - Para agregar valores a un array de forma declarativa podemos usar spread operator.
  • - Para agregar valores a un array de forma declarativa podemos usar spread operator.
  • - No modifica el array original.
Ejemplos:
// Crear constantes, array y objetos const cartOrigin = [];
const product9 = { name: "Smart TV", price: 40000, };
const product10 = { name: "Play Station 5", price: 1000000, };
const product11 = { name: "Tablet", price: 300000, };
console.table(cartOrigin);
// Crear un array para agregar elementos let cartFinal;
cartFinal = [...cartOrigin, product9]
console.table(cartFinal)
// Agregando otro valor al final del array cartFinal = [...cartFinal, product10]
console.table(cartFinal)
// Agregando otro valor al inicio del array cartFinal = [product11, ...cartFinal]
console.table(cartFinal)
console.table('Esto sigue siendo un array vacio porque no se modificó:', cartOrigin)

Clase 37

Eliminar elementos de un array (forma imperativa)
  • - Para eliminar valores a un array de forma imperativa podemos usar métodos como push, pop y splice.
Ejemplos:
// Crear constantes de array y objetos const cartOrigin2 = [];
const product12 = { name: "Iphone 16", price: 1200000, };
const product13 = { name: "Play Station 4", price: 300000, };
const product14 = { name: "Tablet", price: 300000, };
const product15 = { name: "Xiaomi Redmi13", price: 600000, };
cartOrigin2.push(product12, product13, product14, product15);
console.table(cartOrigin2);
// Eliminar ultimo elemento de un array cartOrigin2.pop();
console.table(cartOrigin2);
// Eliminar primer elemento de un array cartOrigin2.shift();
console.table(cartOrigin2);
// Eliminar un elemento de un array segun la posición cartOrigin2.splice(1, 1);
console.table(cartOrigin2);

Clase 38

Destructiring de un array
  • - Un array se puede destructurar al igual que un objeto, pero de diferente forma.
  • - Se obtiene el valor del elemento en base a la posición.
Ejemplos:
// Destructuring de un array, extraer el valor y crear una variable a partir de ese valor const numbers3 = [1, 2, 3, 4];
console.log(numbers3);
const [index0] = numbers3;
// El nombre puede ser cualquiera ya que se accede en orden del index
console.log(index0);
// Si queres una posición diferente sin utilizar las otras posiciones, podes usar comas. const [, , index2] = numbers3;
console.log(index2);
// Si queres obtener el resto de elementos podes usar ... const [, index1, ...rest] = numbers3;
console.log("índice 1 y resto del array", index1, rest);

Clase 39

Bucles forEach y map para recorrer un array
  • - Podemos recorrer un array con el método forEach()
  • - Podemos recorrer un array con el método map(), la diferencia es que crea un array nuevo a partir del que recorre.
Ejemplos:
// Recorriendo un array de productos con forEach() const cart3 = [ { name: "Monitor 24 pulgadas", price: 40000, }, { name: "Monitor 32 pulgadas", price: 60000, }, { name: "Monitor 90 pulgadas", price: 140000, }, { name: "Iphone 15", price: 1110000, }, ];
console.log("Empieza el bucle forEach()");
const newCartForEach = cart3.forEach(function (product) { return `El producto ${product.name} cuesta ARS ${product.price}`; });
console.log(newCartForEach);
// Recorriendo un array de productos con map(|) console.log("Empieza el bucle map()");
const newCartMap = cart3.map(function (product) { return `El producto ${product.name} cuesta ARS ${product.price}`; });
console.log(newCartMap);
