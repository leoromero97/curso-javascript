Clase 17

Operadores mayor y menor qué
  • - Comparar entre números.
  • - Los números pueden ser enteros o flotantes.
Ejemplos:
// Operador mayor qué const operatorNumber1 = 20;
const operatorNumber3 = 30;
console.log("Comparar si un número es mayor a otro", operatorNumber1 > operatorNumber3);
console.log("Comparar si un número es mayor a otro, invirtiendo el orden", operatorNumber3 > operatorNumber1);
// Operador menor qué console.log("Comparar si un número es menor a otro", operatorNumber1 < operatorNumber3);
console.log("Comparar si un número es menor a otro, invirtiendo el orden", operatorNumber3 < operatorNumber1);

Clase 18

Comparación por igualación o negación
  • - Se puede comparar entre cualquier tipo de dato.
  • - Se puede comparar por el valor.
  • - Se puede comparar por el valor y el tipo de dato (modo estricto).
Ejemplos:
// Comparar por igualación: const operatorNumber4 = 20;
const operatorNumber5 = "20";
const operatorNumber6 = 30;
console.log("Compara los valores, ¿Los valores son iguales?", operatorNumber4 == operatorNumber5);
console.log("Comparador los valores y el tipo de dato, ¿Los valores son iguales?", operatorNumber4 === operatorNumber5);
// Comparación por negación: const password1 = "admin";
const password2 = "Admin";
console.log("¿Las contraseñas son diferentes?", password1 != password2);
console.log("¿Los valores son diferentes?", operatorNumber4 != operatorNumber5);
console.log("¿Los valores son diferentes?, modo estricto:", operatorNumber4 !== operatorNumber5);

Clase 19

Null y Undefined
  • - Undefined es un valor que no está declarado o definido todavía.
  • - Null es un valor que no existe.
Ejemplos:
// Undefined let number9;
console.log("El valor no está declarado/definido", number9);
// Null let number10 = null;
console.log("El valor no existe", number10);
