Clase 64

Some
  • - Son métodos que existen para los arrays.
  • - Son soluciones como iteradores pero normalmente sirven como funciones generales en el desarrollo.
Ejemplos:
// Validar si exsite un valor con array de métodos const months4 = [ "Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", "Septiembre" ];
const cart9 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, } ];
months4.forEach((month) => { if (month === "Enero") { console.log("Enero si existe"); } });
const result2 = months4.includes("Enero");
console.log(result2);
const result3 = months4.some((product) => product.name === "Cinta métrica");

Clase 65

FindIndex para encontra la posición de un array
  • - Son métodos que existen para los arrays.
Ejemplos:
// Encontrar la posición de un array const months5 = [ "Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", "Septiembre" ];
const cart10 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, } ];
months5.forEach((month, index) => { if (month === "Marzo") { console.log(`Marzo encontrado en el índice ${index}`); } });
const resultIndex4 = months5.findIndex(month => month === 'Febrero');
console.log(resultIndex4);
const resultIndex5 = cart10.some(product => product.price <= 20000);
console.log(resultIndex5)

Clase 66

reduce() para recibir datos, hacer lógica y entregar un resultado
  • - Son métodos que existen para los arrays.
Ejemplos:
// Ir sumando los precios de los productos y devolver el total a pagar const cart11 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, } ];
let total2 = 0
cart11.forEach(product => total2 += product.price)
console.log(total2)
let result6 = cart11.reduce((total2, product) => total2 + product.price, 0)
console.log(result6);

Clase 67

filter() crea un nuevo array filtrado
  • - Son métodos que existen para los arrays.
Ejemplos:
// Filtrar un objeto de productos const cart12 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, }, { name: "Sierra circular", price: 30000, }, { name: "Lijadora orbital", price: 25000, }, { name: "Martillo", price: 10000, }, ];
let result7;
result7 = cart12.filter((product) => product.price >= 20000);
result7 = cart12.filter((product) => product.name !== "Martillo");
console.log(result7);

Clase 68

find() crea un nuevo array en base a la condición
  • - Son métodos que existen para los arrays.
  • - Retorna el primer elemento que cumpla la condición.
Ejemplos:
// Encontrar un elemento de un array y retornar un array nuevo const cart13 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, }, { name: "Sierra circular", price: 30000, }, { name: "Lijadora orbital", price: 25000, }, { name: "Martillo", price: 10000, }, ];
// con forEach()
let result8 = '';
cart13.forEach((product, index) => { if (product.name === 'Martillo') { result8 = cart13[index] } })
let result9 = cart13.find(product => product.name === 'Martillo')
console.log(result9);

Clase 69

every() crea un nuevo array en base a la condición
  • - Son métodos que existen para los arrays.
  • - Todos los elementos que valide dentro del array deben cumplir la condición.
Ejemplos:
// Retorna false porque no cumplen todos con la condición const cart14 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, }, { name: "Sierra circular", price: 30000, }, { name: "Lijadora orbital", price: 25000, }, { name: "Martillo", price: 10000, }, ];
const result10 = cart14.every((product) => product.price > 20000);
console.log(result10);

Clase 70

concat() une dos arrays
  • - Son métodos que existen para los arrays.
  • - El orden es importante.
Ejemplos:
// Concatenar arrays const months6 = [ "Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", ];
const months7 = ["Septiembre", "Octubre", "Noviembre"];
const months8 = ["Diciembre"];
const allMonths1 = months6.concat(months7, months8);
console.log(allMonths1);
const allMonths2 = [...months6, ...months7, ...months8, "Mes desconocido"];
console.log(allMonths2);

Clase 71

Spread Operator
  • - Son métodos que existen para los arrays.
  • - El orden es importante.
  • - Programación funcional al usar spread operator en arrays.
Ejemplos:
Spread operator const months9 = [ "Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio", "Agosto", "Septiembre", ];
const cart15 = [ { name: "Caja de herramientas 16 pulgadas", price: 20000, }, { name: "Taladro inalámbrico", price: 45000, }, { name: "Juego de destornilladores", price: 15000, }, ];
console.log(months9);
const months10 = [...months9, "Ocutbre", "Noviembre"];
console.log(months10);
const product16 = { name: "Sierra circular", price: 30000 };
const cart16 = [...cart15, product16];
console.log(cart16);
