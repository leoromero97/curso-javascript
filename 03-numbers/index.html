Clase 11

Fundamentos de tipo de dato Numbers o números
  • - Es un número, puede ser entero, negativo o flotante. El tipo siempre va a ser Number.
  • - Se pueden realizar operaciones matemáticas entre valores de tipo number.
  • - Tienen métodos nativos de JavaScript.
Ejemplos:
// Inicializar variables con un valores de tipo number: const number1 = 20;
console.log(number1); const number6 = Number(80);
console.log(number6); const number7 = new Number(70);
console.log(number7);
// Números flotantes o negativos: const number3 = 20.2;
console.log(number3); const number4 = 0.12312131;
console.log(number4); const number5 = -10;
console.log(number5);

Clase 12

Operaciones con numbers
  • - Suma, resta, multiplicación, división y módulo son las operaciones más comunes de los tipos de datos numbers.
Ejemplos:
// Suma: const number8 = 20
const number9 = 10
const additionResult = number8 + number9
console.log('Suma', additionResult)
// Resta: const subtractionResult = number8 - number9
console.log('Resta', subtractionResult)
// Multiplicación: const multiplicationResult = number8 * number9
console.log('Multiplicación', multiplicationResult)
// División: const divisionResult = number8 / number9
console.log('División', divisionResult)
// Módule: const moduleResult = number8 % number9
console.log('Módulo', moduleResult)

Clase 13

El objeto Math para otras operaciones
  • - Tiene propiedades y métodos definidos de forma nativa para operaciones más complejas como obtener el valor de PI.
  • - Se pueden combinar los métodos como más útil sea, por ejemplo obtener un número random en un rango.
Ejemplos:
// El resultado de PI console.log(Math.PI);
// Redondear números console.log("Redondear", Math.round(2.4));
console.log("Redondear hacia arriba", Math.ceil(2.2));
console.log("Redondear hacia abajo", Math.floor(2.7));
// Operaciones complejas console.log("Raíz cuadrada", Math.sqrt(144));
console.log("Valor absoluto", Math.abs(500));
console.log("Potencia", Math.pow(2, 4));
// Operaciones útiles para el día a día console.log("Conocer el número mínimo", Math.min(4, 7, -1, 12));
console.log("Conocer el número máximo", Math.max(4, 7, 1, 12));
console.log("Número aleatorio entre 0 y 100", Math.floor(Math.random() * 100));

Clase 14

El orden de las operaciones
  • - Es igual que en matemáticas.
  • - Se puede utilizar paréntesis, corchetes o llaves para separar en términos.
Ejemplos:
// Orden por default const product4WithSpaces = console.log("Primero 30 * 2 y luego suma 20", 20 + 30 * 2);

Clase 15

Incrementos y decrementos
  • - Se puede incrementar o decrementar de a uno.
  • - Se puede incrementar o decrementar de a varios valores de tipo number.
  • - Se utiliza ++ o --.
Ejemplos:
// Incrementar let puntaje1 = 5;
console.log(puntaje1);
console.log("Incrementar de a uno, primero ejecuta el valor inicial y luego incrementa", puntaje1++);
console.log("Incrementar de a uno, primero incrementa y luego muestra el valor", ++puntaje1);
// Decrementar console.log("Decrementar de a uno, primero ejecuta el valor inicial y luego incrementa", puntaje1--);
console.log("Decrementar de a uno, primero incrementa y luego muestra el valor", --puntaje1);
// Incrementar de a más de a uno const pointRugby = 3;
puntaje1 += pointRugby;
console.log("Se le suma 3 al puntaje inicial", puntaje1);

Clase 16

Convertir strings a numbers
  • - Los métodos están en el objeto Number (palabra reservada).
  • - Podemos convertir un string a un número entero o flotante.
  • - Podemos saber si un número es entero o no.
Ejemplos:
// Conocer el tipo de dato const numberStringOne = "20";
console.log("Primer constante: ", numberStringOne);
console.log("Tipo de dato de la primer constante:", typeof numberStringOne);
// Convertir un string a un número entero console.log("Ahora el string es un número: ", Number.parseInt(numberStringOne));
// Convertir un string a un número flotante const numberStringTwo = "20.2";
console.log(numberStringTwo);
console.log( "Ahora el string es un número flotante: ", Number.parseFloat(numberStringTwo) );
// Validar si un número es entero o no const numberStringThree = "Uno";
console.log(numberStringThree);
console.log("¿Es un número entero?", Number.isInteger(numberStringThree));

const numberStringFour = 20;
console.log(numberStringFour);
console.log("¿Es un número entero?", Number.isInteger(numberStringFour));
