Clase 20

Valores de booleans
  • - Solo tiene 2 valores, true o false.
Ejemplos:
// Asignando valores booleanos const boolean1 = true;
const boolean2 = false;
console.log(boolean1);
console.log(boolean2);
console.log(typeof boolean1);
// Comparando 2 valores de tipo boolean: console.log("Comparar 2 valores booleans:", boolean1 === boolean2);
// Crear un constructor de tipo boolean: const boolean3 = new Boolean(true);
console.log(boolean3);

Clase 21

Operador ternario
  • - Una forma más simplificada de validar por si o por no.
Ejemplos:
// Operador ternario const isAuth = true;
console.log(isAuth ? "Si está autenticado" : "No está autenticado");
